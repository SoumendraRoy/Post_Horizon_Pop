### Do We Recover the Prior Distribution Beyond the Detection Horizon in Gravitational-Wave Population Analysis?

To answer this question, we construct a simple normal--normal model. We draw $N$ samples $x$ from a distribution
\[
x \sim \mathcal{N}(\mu, \sigma).
\]
We then add parameter-estimation noise such that the observed values satisfy
\[
x_o \sim \mathcal{N}(x, \sigma_o).
\]
Finally, we apply a detection threshold by selecting only events with
\[
x_o > a,
\]
where $a$ represents the detection horizon.
