### Do We Recover the Prior Distribution Beyond the Detection Horizon in Gravitational-Wave Population Analysis?

To answer this question, we construct a simple normal–normal model. We draw N samples x from a distribution x ~ N(μ, σ). We then add parameter-estimation noise such that the observed values satisfy xo ~ N(x, σo). Finally, we apply a detection threshold by selecting only events with xo > a, where a represents the detection horizon.
